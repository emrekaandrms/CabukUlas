# CabukUlas Admin Web

CabukUlas icin auth-gated web admin paneli.

## Kurulum

```bash
cp .env.example .env.local
npm install
npm run dev
```

Gerekli env degiskenleri:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## Moduller

- Overview
- Companies
- Categories
- Contact Channels
- Review Queue
- Placements
- Data Health
- Reports
- Analytics
- Audit Log
- Settings
